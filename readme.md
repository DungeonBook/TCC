DungeonBook: 

As comunidades de jogadores de RPG (Role-Playing Game) podem enfrentar dificuldades para organizar suas partidas, seja pela falta de jogadores disponíveis ou pela ausência de plataformas digitais especializadas em agendamento de salas. Diante desse cenário, desenvolvemos um sistema web completo para criação e agendamento de salas de RPG.
O sistema implementado integra funcionalidades como cadastro de usuários, gerenciamento de perfis, visualização de salas disponíveis para participar. Com isso, a ferramenta mostrou-se eficaz para otimizar a organização das partidas de RPG, contribuindo para a ampliação e fortalecimento das comunidades dedicadas a esse tipo de jogo colaborativo.
